## How to Generate Your Own Reports

1. **Set up your environment**:
   ```bash
   cp .env.example .env
   # Edit .env with your AWS configuration
   ```

2. **Run the report generator**:
   ```bash
   uv run python cur_report_generator.py
   ```

3. **Find your reports**:
   ```bash
   open reports/cur_report_*.html
   ```
